Argumen

post /login

Login untuk mendapatkan autentikasi. Autentikasi berbasis dengan JWT token. Jika belum membuat user maka beralih ke /register

Argumen

nama deskripsi tipe
email berisikan alamat email valid string
password berisikan password string

Contoh Response

Berhasil login!
dan mengembalikan jwt cookie

post /logout

untuk menghapus cookie autentikasi

Contoh Response

berhasil logout!

post /register

Login untuk mendapatkan autentikasi. Autentikasi berbasis dengan JWT token. Jika belum membuat user maka beralih ke /register.

Argumen

nama deskripsi tipe
nama Berisikan nama lengkap dengan maksimal 128 karakter string
email berisikan alamat email valid maksimal 255 karakter string
password berisikan password bebas string
telepon berisikan nomor telepon valid (tanpa karakter) string

Contoh Response

kode status isi alasan
400 email sudah terdaftarkan! email sudah ada di dalam database (duplikat)
400 Input tidak lengkap! Salah satu input belum terisi
400 Input tidak valid! panjang karakter melebihi batas
200 Sukses mendaftarkan pengguna valid dan benar
